
# Projeto de Análise de Dados - Alura Store

Este projeto analisa o desempenho de quatro lojas fictícias para ajudar o Sr. João
a decidir qual loja vender para investir em um novo negócio.

## Análises realizadas

- Faturamento total por loja
- Vendas por categoria
- Produtos mais e menos vendidos
- Frete médio por loja

## Tecnologias

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Objetivo

Identificar a loja com pior desempenho considerando faturamento,
avaliações e custos de frete.
